Ejercicio 7 — Inicialización de sesión

Este ejercicio inicia una sesión en el servidor y muestra el ID de sesión.

Iniciar sesión y ver ID (ejecutar PHP)

Notas:

  • Usa un servidor PHP (por ejemplo: php -S localhost:8000).
  • Si ya existe una sesión activa en el navegador, verás el mismo ID mientras la sesión esté viva.
