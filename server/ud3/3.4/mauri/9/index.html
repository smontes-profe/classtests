Ejercicio 9 - Verificar Acceso

Este ejercicio simula un sistema de login usando sesiones.

Iniciar Sesión Verificar Acceso

Funcionamiento:

  • Inicia sesión para establecer tu rol como "Administrador"
  • Usa "Verificar Acceso" para comprobar si tienes permisos
  • Si no has iniciado sesión, verás "Acceso denegado"
  • Si has iniciado sesión, verás "Acceso concedido como: Administrador"
