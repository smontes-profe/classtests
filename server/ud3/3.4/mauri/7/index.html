Ejercicio 7 - Inicialización de Sesión

Este ejercicio permite establecer y mostrar un ID de sesión personalizado.

Gestionar ID de Sesión

Funcionamiento:

  • Podrás ver el ID de sesión actual y cambiarlo si lo deseas
  • Introduce el ID que prefieras en el formulario
  • El ID se mantendrá mientras no cierres el navegador
  • Puedes refrescar la página para comprobar que el ID se mantiene
