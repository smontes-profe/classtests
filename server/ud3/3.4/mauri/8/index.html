Ejercicio 8 - Guardar en Sesión

Este ejercicio guarda un valor en la sesión y muestra tanto el ID como el valor guardado.

Gestionar Sesión

Funcionamiento:

  • Se iniciará una sesión con tu ID personalizado
  • Se guardará el rol "Administrador" en la sesión
  • Podrás ver tanto el ID como el rol almacenado
  • Los datos se mantienen mientras no cierres el navegador
