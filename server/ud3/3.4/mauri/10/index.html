Ejercicio 10 - Destruir Sesión

Este ejercicio permite destruir completamente una sesión activa.

Iniciar Sesión Verificar Acceso

Funcionamiento:

  • Inicia sesión para establecer tu rol como "Administrador"
  • Usa "Verificar Acceso" para comprobar si tienes permisos
  • Si estás logueado, verás un botón para destruir la sesión
  • Al destruir la sesión, volverás a ver "Acceso denegado"
